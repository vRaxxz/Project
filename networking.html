Computer Networking

Network

A network is a group of devices (like computers and phones) connected together so they can share information and resources, like files or printers.

IP Address

An IP address is a unique number assigned to each device in a network to identify it and allow communication over the Internet or local network.

Devices

Router

Connect your home network and or any network to the internet. It manages data traffic between devices and the internet.

Wireless Access Point

Connects wireless devices to a wired network and can extend the network's reach.

Switch

Connect devices to a local network allowing them to communicate to each other.

5G vs 2.4G WiFi

5G is faster but shorter range and it is good for streaming and gaming.
2.4G slower but longer range.

How to Change WiFi Name and Password

Step by step:
on your keyboard key click the windows icon + R,
type in cmd and click enter.
write the command ipconfig and then you'll see a list of IP Addresses.
find the name where it says Default Gateway.
Copy the address the paste it in your browser.
Login and change the wifi settings.
